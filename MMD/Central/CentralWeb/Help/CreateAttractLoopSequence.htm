How To Create an Attract Loop Sequence:

1. Go to the Add an Attract Loop Sequence Page
[Manage Attract Loops > Manage Attract Loops > Add an Attract Loop Sequence]

2. Specify the Loop Sequence Name and Store Concept.
Click [Submit Changes to Attract Loop Sequence] to save the name and concept.

3a. To add a Product List to the Attract Loop click [Add Product List sequence to this Attract Loop]
Specify the Loop Item Name [it will be automatically named if left blank]
Specify the Show Time in seconds
Select the Product Recommend List Type
Click [Add Attract Loop Item]
You will now see this item in your Attract Loop Sequence

3b. To add an individual product to the Attract Loop click [Add a Single Product to this Attract Loop]
Specify the Loop Item Name [it will be automatically named if left blank]
Specify the Show Time in seconds
Specify the Item ID for the product you want to display
Click [Add Attract Loop Item]
You will now see this item in your Attract Loop Sequence

3c. To add an uploaded file to the Attract loop, click [Add] next to the Attract Loop Item listed at the bottom of the page.
For Video: Specify the start and end time. If left blank, it will play the entire clip.
For Image: Specify the show time in seconds. If you want the item to link to a specific page, insert the link URL.
For ScreenML: Specify the show time in seconds. If you want to specify arguments for that file, insert the query string.

4. When you are finished adding Attract Loop Sequence Items, click [Submit Changes to Attract Loop Sequence]

5. To submit the changes, go to the Manage Attract Loops page and click [Submit All Changes]
