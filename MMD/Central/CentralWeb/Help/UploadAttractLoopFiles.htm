How To Upload Attract Loop Files:

1. Go to the View Attract Loop Items page
[Manage Attract Loops > Manage Attract Loop Files]

2. Click [Add an Attract Loop Item]

3. Specify the friendly name of the loop item to be used in the system. [This is not displayed in the application.]
Specify the Store Concept.
Specify the Attract Loop Item Type. [Videos must be .asx or .wmv] [images must be 640×480] [ScreenML must be valid ScreenML]
Click [Browse] to select the file for upload.

4. Click [Create this Attract Loop Item]

5. Repeat for all files to be added as Attract Loop Items.
