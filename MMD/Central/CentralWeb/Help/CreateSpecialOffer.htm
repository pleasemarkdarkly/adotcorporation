How To Create a Special Offer List:

1.Upload the image files associated with the special offers. This includes the list image and the detail image.
[Manage Recommend Lists > Manage Recommend List Files > Add New Recommend List File]

2. Specify the Friendly name for the file to be used in the system, which concept(s) the file will be used for, and then specify the file to be uploaded. Repeat this for all known files. Additional files can be uploaded at any time.

3. Create a New Promotion Recommend List
[Manage Recommend Lists > Promotion Recommend Lists]

4. Specify the Concept and [Special Offer] List Type then click [Create New Promotion Recommend List]

5. Specify the [Promotion ID] (make one up) and click [Add].
Specify the Promotion Name.
Specify the Special Offer Detail text.
The Link Image is the file you uploaded to be displayed in the Special Offer list.
The Detail Image is the file you uploaded to be displayed in the Special Offer Detail page with the detail test.
Click [Update List Attribute Changes] to submit the Special Offer Item.
Click [Back to Edit [Special Offers] Recommendation List ] to add more items
Repeat for all Special Offers in in the list.

6. Click [Back to [Promotion] Recommendation List] to Submit your changes.
Click [Submit] or [Submit All Changes] to commit the changes.
